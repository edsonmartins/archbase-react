## Eventos úteis

### onResizeStart

Acionado quando um redimensionamento do mouse/toque é iniciado. A ação de redimensionamento pode ser cancelada retornando `false` do manipulador de eventos.

#### Aplica-se a

`<ArchbaseSpaceLeftResizable />` `<ArchbaseSpaceTopResizable />` `<ArchbaseSpaceRightResizable />` `<ArchbaseSpaceBottomResizable />`

#### Parâmetros

- **resizeType**: "resize-left" | "resize-top" | "resize-right" | "resize-bottom" - Type of resize operation started

#### Exemplo

```tsx
const Exemplo = () => 
	<ArchbaseSpaceLeftResizable 
		onResizeStart={(resizeType) => /* lógica para determinar se o redimensionamento deve ser possível */}>
	...
	</ArchbaseSpaceLeftResizable>
```

### onResizeEnd

Disparado ao final de uma ação de redimensionamento fornecendo o novo tamanho (em pixels) do espaço.

#### Aplica-se a

`<ArchbaseSpaceLeftResizable />` `<ArchbaseSpaceTopResizable />` `<ArchbaseSpaceRightResizable />` `<ArchbaseSpaceBottomResizable />`

#### Parâmetros

- **newSize**: number - Novo tamanho de espaço. Para `<ArchbaseSpaceLeftResizable />` e `<ArchbaseSpaceRightResizable />` 
esta será a largura, por `<ArchbaseSpaceTopResizable />` e `<ArchbaseSpaceBottomResizable />` essa será a altura
- **domRect**: DOMRect - Dimensões finais do espaço
- **resizeType**: "resize-left" | "resize-top" | "resize-right" | "resize-bottom" - Tipo de redimensionamento
operação iniciada

#### Exemplo

O cenário provável em que você precisa usar isso é manter o tamanho do espaço controlado
no estado:

```tsx
const Exemplo = () => {
	const [ size, setSize ] = useState(200);

	return (
		<ArchbaseSpaceViewPort>
			<ArchbaseSpaceLeftResizable 
				onResizeEnd={(newSize, domRect, resizeType) => setSize(newSize)}>
			  ...
			</ArchbaseSpaceLeftResizable>
		</ArchbaseSpaceViewPort>
	)
}
```

### Eventos de elemento

Os espaços permitem que você se vincule aos seguintes eventos do componente React.

- `onClick`
- `onDoubleClick`
- `onMouseDown`
- `onMouseEnter`
- `onMouseLeave`
- `onMouseMove`
- `onTouchStart`
- `onTouchMove`
- `onTouchEnd`

#### Aplica-se a

`<ArchbaseSpaceFill />` `<ArchbaseSpaceLeft />` `<ArchbaseSpaceTop />` `<ArchbaseSpaceRight />` `<ArchbaseSpaceBottom />` `<ArchbaseSpaceLeftResizable />` `<ArchbaseSpaceTopResizable />` `<ArchbaseSpaceRightResizable />` `<ArchbaseSpaceBottomResizable />` `<ArchbaseSpacePositioned />` `<ArchbaseSpaceCustom />`
