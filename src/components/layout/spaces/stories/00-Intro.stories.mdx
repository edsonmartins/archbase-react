## Introdução

**Um sistema de layout aninhado e fácil de entender, Archbase Spaces permite dividir uma página
ou contêiner em espaços ancorados, roláveis ​​e redimensionáveis, permitindo que você crie interfaces 
de usuário desktop no navegador.**

Archbase Spaces permite dividir um elemento HTML de página ou contêiner em espaços. Esses
espaços sabem como se comportar uns em relação aos outros e também podem ser divididos em outros
espaços aninhados.

A biblioteca contém um pequeno número de componentes que podem ser usados ​​para construir
interfaces de usuário do tipo desktop. Veja a seguir os layouts que podem ser construídos:
