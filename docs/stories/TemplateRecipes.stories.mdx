# Template Recipes

Ao montar uma página (FormTemplate ou SpaceTemplate) use sempre o `canonicalUrl` do catálogo para referenciar a story oficial.

```
{
  "name": "ArchbaseFormTemplate",
  "canonicalUrl": "https://docs.archbase.com/storybook/archbase-form-template"
}
```

<Story name="Links">
  {() => (
    <ul>
      {['ArchbaseFormTemplate', 'ArchbaseSpaceTemplate']
        .map((name) => catalog.components.find((c) => c.name === name))
        .filter(Boolean)
        .map((component) => (
          <li key={component?.name}>
            <strong>{component?.name}</strong>: <a href={component?.canonicalUrl}>{component?.canonicalUrl}</a>
          </li>
        ))}
    </ul>
  )}
</Story>

Use esses links dentro de MDX/recipes e cite o catálogo sempre que atualizar as props/estados. Essa base alimenta `llms.txt`, `DOCS_AUDIT.md` e o portal editorial.
