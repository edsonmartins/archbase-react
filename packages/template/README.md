# @archbase/template

Template components for Archbase React v3 - High-level templates and layouts.

## Installation

```bash
npm install @archbase/template
# or
pnpm add @archbase/template
# or
yarn add @archbase/template
```

## Features

- **CRUD Templates**: Pre-configured CRUD page templates
- **Form Templates**: Form page templates
- **List Templates**: Data list page templates
- **Detail Templates**: Detail view templates
- **Dashboard Templates**: Dashboard layout templates

## Dependencies

```json
{
  "@mantine/core": "8.3.12",
  "@mantine/hooks": "8.3.12",
  "@mantine/modals": "8.3.12",

  "react-dom": "^18.3.0 || ^19.0.0"
}
```

## Usage

```typescript
import { 
  CrudPageTemplate,
  FormPageTemplate,
  ListPageTemplate 
} from '@archbase/template';
```
