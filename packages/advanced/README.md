# @archbase/advanced

Advanced components for Archbase React - Charts, Kanban, QueryBuilder.

## Installation

```bash
npm install @archbase/advanced
# or
pnpm add @archbase/advanced
# or
yarn add @archbase/advanced
```

## Features

- **Charts**: Chart.js and ApexCharts integration
- **QueryBuilder**: Visual query builder component
- **Kanban**: Kanban board component
- **Advanced Forms**: Complex form components
- **Multi-select**: Advanced multi-select components
- **Date Pickers**: Advanced date selection components

## Dependencies

```json
{
  "@mantine/core": "8.3.12",
  "@mantine/hooks": "8.3.12",
  "@tabler/icons-react": "^3.29.0",
  "react": "^19.0.0",
  "react-dom": "^19.0.0"
}
```

## Usage

```typescript
import { ChartBuilder, QueryBuilder } from '@archbase/advanced';
```
