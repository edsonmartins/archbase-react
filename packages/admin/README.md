# @archbase/admin

Admin interface components for Archbase React.

## Installation

```bash
npm install @archbase/admin
# or
pnpm add @archbase/admin
# or
yarn add @archbase/admin
```

## Features

- **Admin Layout**: Pre-configured admin layout components
- **Navigation**: Sidebar and navigation components
- **User Management**: User profile and settings components
- **Flag Management**: Feature flag management UI

## Dependencies

This package requires the following peer dependencies:

```json
{
  "@mantine/core": "8.3.12",
  "@mantine/hooks": "8.3.12",
  "@tabler/icons-react": "^3.27.0",
  "react": ">=18.0.0",
  "react-dom": ">=18.0.0"
}
```
