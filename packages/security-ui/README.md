# @archbase/security-ui

Security UI components for Archbase React - Forms, Modals, and Views.

## Installation

```bash
npm install @archbase/security-ui
# or
pnpm add @archbase/security-ui
# or
yarn add @archbase/security-ui
```

## Features

- **Login Forms**: Pre-built login form components
- **Auth Modals**: Authentication modal dialogs
- **Password Recovery**: Password reset components
- **User Profile**: User profile editing components
- **Role Management**: UI for managing user roles

## Dependencies

```json
{
  "@mantine/core": "8.3.12",
  "@mantine/hooks": "8.3.12",
  "@mantine/modals": "8.3.12",
  "@tabler/icons-react": "^3.27.0",
  "react": ">=18.0.0",
  "react-dom": ">=18.0.0"
}
```

## Usage

```typescript
import { 
  LoginForm,
  PasswordRecoveryModal,
  UserProfileForm 
} from '@archbase/security-ui';
```
