# Server-Side Rendering

Utilitários para SSR com TanStack Start/Router.

## Instalação

```bash
pnpm add @archbase/ssr
```

## Configuração

```tsx
import { ArchbaseSSRProvider } from '@archbase/ssr';

function App() {
  return (
    <ArchbaseSSRProvider>
      <MyApp />
    </ArchbaseSSRProvider>
  );
}
```
