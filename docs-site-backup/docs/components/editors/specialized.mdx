# Componentes Especializados

Componentes para casos de uso específicos.

## ArchbaseColorEdit

Seletor de cores.

### Exemplo

```tsx
import { ArchbaseColorEdit } from '@archbase/components';

<ArchbaseColorEdit
  dataSource={dataSource}
  dataField="corPrincipal"
  label="Cor Principal"
  format="hex"
  disabled={isBrowsing}
/>
```

### Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `format` | `'hex' \| 'rgb' \| 'rgba' \| 'hsl'` | `'hex'` | Formato da cor |
| `swatches` | `string[]` | - | Cores predefinidas |
| `withPicker` | `boolean` | `true` | Mostra color picker |

## ArchbaseAvatarEdit

Upload e edição de avatar.

### Exemplo

```tsx
import { ArchbaseAvatarEdit } from '@archbase/components';

<ArchbaseAvatarEdit
  dataSource={dataSource}
  dataField="foto"
  label="Foto do Perfil"

  disabled={isBrowsing}
/>
```

### Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `size` | `number` | `100` | Tamanho em pixels |
| `allowCrop` | `boolean` | `true` | Permite recorte |
| `accept` | `string` | `'image/*'` | Tipos aceitos |

## ArchbaseFileUpload

Upload de arquivos com progresso.

### Exemplo

```tsx
import { ArchbaseFileUpload } from '@archbase/components';

<ArchbaseFileUpload
  dataSource={dataSource}
  dataField="anexos"
  label="Anexos"
  multiple
  maxSize={10 * 1024 * 1024} // 10MB
  accept={['.pdf', '.doc', '.docx']}
  disabled={isBrowsing}
/>
```

### Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `multiple` | `boolean` | `false` | Múltiplos arquivos |
| `maxSize` | `number` | - | Tamanho máximo em bytes |
| `maxFiles` | `number` | - | Número máximo de arquivos |
| `accept` | `string[]` | - | Extensões aceitas |

## ArchbaseDropzone

Área de drop para arquivos.

### Exemplo

```tsx
import { ArchbaseDropzone } from '@archbase/components';

<ArchbaseDropzone
  onDrop={(files) => console.log('Arquivos:', files)}
  accept={['image/*']}
  maxSize={5 * 1024 * 1024}
>
  <p>Arraste imagens aqui ou clique para selecionar</p>
</ArchbaseDropzone>
```
