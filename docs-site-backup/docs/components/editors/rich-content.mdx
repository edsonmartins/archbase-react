# Conteúdo Rico

Componentes para edição de conteúdo complexo.

## ArchbaseRichTextEdit

Editor de texto rico com formatação.

### Exemplo

```tsx
import { ArchbaseRichTextEdit } from '@archbase/components';

<ArchbaseRichTextEdit
  dataSource={dataSource}
  dataField="descricao"
  label="Descrição"
  disabled={isBrowsing}
/>
```

### Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `dataSource` | `ArchbaseDataSource<T>` | - | Fonte de dados |
| `dataField` | `string` | - | Campo do objeto |
| `label` | `string` | - | Rótulo |
| `minHeight` | `number` | `200` | Altura mínima em pixels |
| `maxHeight` | `number` | - | Altura máxima |
| `toolbar` | `string[]` | - | Botões da toolbar |

### Toolbar Disponível

- `bold`, `italic`, `underline`, `strike`
- `h1`, `h2`, `h3`, `h4`
- `bulletList`, `orderedList`
- `link`, `image`
- `blockquote`, `code`, `codeBlock`
- `alignLeft`, `alignCenter`, `alignRight`

## ArchbaseJsonEdit

Editor de JSON com syntax highlighting.

### Exemplo

```tsx
import { ArchbaseJsonEdit } from '@archbase/components';

<ArchbaseJsonEdit
  dataSource={dataSource}
  dataField="configuracoes"
  label="Configurações JSON"
  disabled={isBrowsing}
/>
```

### Props Específicas

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `validateOnBlur` | `boolean` | `true` | Valida JSON ao sair do campo |
| `formatOnBlur` | `boolean` | `true` | Formata JSON ao sair do campo |
| `indentSize` | `number` | `2` | Tamanho da indentação |
