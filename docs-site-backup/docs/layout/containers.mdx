# Containers

Componentes de container para estruturação de conteúdo.

## ArchbaseCard

Card com seções estruturadas.

```tsx
import { ArchbaseCard } from '@archbase/layout';

<ArchbaseCard
  title="Título do Card"
  subtitle="Subtítulo opcional"
  actions={<Button>Ação</Button>}
>
  Conteúdo do card
</ArchbaseCard>
```

## ArchbaseContainer

Container responsivo centralizado.

```tsx
import { ArchbaseContainer } from '@archbase/layout';

<ArchbaseContainer size="md" padding="lg">
  Conteúdo centralizado
</ArchbaseContainer>
```

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'` | `'md'` | Largura máxima |
| `padding` | `string` | `'md'` | Padding interno |
