# Getting Started

Para iniciar com Archbase React:

1. Inicie o `ArchbaseGlobalProvider` com `translationName`, `themeLight`/`themeDark` e o `containerIOC`.
2. Configure `initArchbaseI18nEarly` *antes* de renderizar, garantindo que `useArchbaseTranslation` tenha namespaces prontos.
3. Use componentes base (`ArchbaseEdit`, `ArchbaseActionButtons`, `ArchbaseDataGrid`) cujo `canonicalUrl` aparece no catálogo:

```json
{
  "name": "ArchbaseEdit",
  "canonicalUrl": "https://docs.archbase.com/storybook/archbase-edit"
}
```

4. Combine com os recipes do Storybook (veja `docs/stories/FormRecipes.stories.mdx` e `docs/stories/DataGridRecipes.stories.mdx`) para ver exemplos executáveis.

Importante: rode `pnpm generate:catalog` após mudanças nos componentes para manter `llms.txt`, docs e receitas alinhados.
