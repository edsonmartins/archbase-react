# Templates

Templates CRUD (Form, Space, Grid) unem layout + data source + actions:

- `ArchbaseFormTemplate`: formulário completo com toolbar e validação. Use `dataSource`, `renderForm`/`renderToolbar`.
- `ArchbaseSpaceTemplate`: estrutura página com filtros, toolbar e grid. Ideal para dashboards com `filters`, `content`, `toolbar`.
- `ArchbaseGridTemplate`: combina DataGrid + filtros + actions rápidas para listas.

## Padrões

1. **Ação centralizada**: use `ArchbaseActionButtons` para salvar/cancelar; configure `isSaving`/`isCanceling`.
2. **Filtros**: insira `ArchbaseSimpleFilter` ou `ArchbaseAdvancedFilter` no `filterComponent`.
3. **Empty/loading**: defina `emptyState`, `loading` e `notifications`.
4. **Segurança**: combine com `ArchbaseSecureFormField` para esconder campos quando falta permissão (link do catálogo abaixo).
