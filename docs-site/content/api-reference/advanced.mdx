
# @archbase/advanced

Componentes avançados de consulta e filtros.

## Componentes

| Componente | Descrição |
|------------|-----------|
| `ArchbaseQueryBuilder` | Construtor de queries |
| `ArchbaseAdvancedFilter` | Filtro avançado |
| `QueryFields` | Container de campos |
| `QueryField` | Definição de campo |
| `QueryFieldValue` | Valor predefinido |

## Utilitários

| Função | Descrição |
|--------|-----------|
| `getDefaultEmptyFilter` | Filtro vazio |
| `filterToRSQL` | Converte para RSQL |
| `filterToSQL` | Converte para SQL |

## Constantes

| Constante | Descrição |
|-----------|-----------|
| `OP_EQUALS` | Operador igual |
| `OP_NOT_EQUALS` | Operador diferente |
| `OP_CONTAINS` | Operador contém |
| `OP_STARTS_WITH` | Operador inicia com |
| `OP_ENDS_WITH` | Operador termina com |
| `OP_GREATER_THAN` | Operador maior que |
| `OP_LESS_THAN` | Operador menor que |
| `OP_BETWEEN` | Operador entre |
