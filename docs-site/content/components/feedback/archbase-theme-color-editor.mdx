# ArchbaseThemeColorEditor

Editor visual de temas Mantine para criar e customizar temas em tempo de execução.
Permite editar cores, tipografia, espaçamento e bordas com preview ao vivo.

## Uso Básico

```tsx
import { ArchbaseThemeColorEditor } from '@archbase/components';

function App() {
  const [theme, setTheme] = useState({});

  return (
    <ArchbaseThemeColorEditor
      theme={theme}
      onChange={setTheme}
      onApply={(newTheme) => console.log('Tema aplicado:', newTheme)}
    />
  );
}
```

## Tema Inicial

```tsx
import { ArchbaseThemeColorEditor } from '@archbase/components';

function App() {
  const [theme, setTheme] = useState({
    colors: {
      primary: '#3B82F6',
      secondary: '#8B5CF6',
      success: '#10B981',
      warning: '#F59E0B',
      error: '#EF4444'
    }
  });

  return (
    <ArchbaseThemeColorEditor
      theme={theme}
      onChange={setTheme}
    />
  );
}
```

## Seções Customizadas

```tsx
import { ArchbaseThemeColorEditor } from '@archbase/components';

function App() {
  return (
    <ArchbaseThemeColorEditor
      theme={theme}
      onChange={setTheme}
      sections={['colors', 'typography', 'spacing']}
      defaultSection="colors"
    />
  );
}
```

### Seções Disponíveis

- `'colors'` - Editor de cores
- `'typography'` - Editor de tipografia
- `'spacing'` - Editor de espaçamento
- `'borders'` - Editor de bordas

## Exportação de Tema

```tsx
import { ArchbaseThemeColorEditor, type ExportFormat } from '@archbase/components';

function App() {
  const [exportFormat, setExportFormat] = useState<ExportFormat>('json');

  return (
    <ArchbaseThemeColorEditor
      theme={theme}
      onChange={setTheme}
      exportFormat={exportFormat}
      onExport={(data) => {
        console.log('Tema exportado:', data);
      }}
    />
  );
}
```

### Formatos de Exportação

- `'json'` - Objeto JSON
- `'typescript'` - Definição de tipo TypeScript
- `'css'` - Variáveis CSS
- `'mantine'` - Configuração Mantine completa

## Presets


## Funções Auxiliares

```tsx
import {
  generateThemeFromColors,
  getDefaultMantineTheme,
  mergeWithDefaultTheme
} from '@archbase/components';

function App() {
  // Gerar tema completo a partir de cores primárias
  const theme = generateThemeFromColors({
    primary: '#3B82F6',
    secondary: '#8B5CF6'
  });

  // Obter tema padrão do Mantine
  const defaultTheme = getDefaultMantineTheme();

  // Mesclar com tema padrão
  const mergedTheme = mergeWithDefaultTheme(theme);
}
```

## Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `theme` | `MantineThemeOverride` | - | Tema atual |
| `onChange` | `function` | - | Callback ao alterar tema |
| `onApply` | `function` | - | Callback ao aplicar tema |
| `onExport` | `function` | - | Callback ao exportar tema |
| `onReset` | `function` | - | Callback ao resetar tema |
| `showPreview` | `boolean` | `true` | Mostrar preview ao vivo |
| `previewPosition` | `'left' \| 'right'` | `'right'` | Posição do preview |
| `readOnly` | `boolean` | `false` | Modo apenas leitura |
| `sections` | `string[]` | `all` | Seções a mostrar |
| `defaultSection` | `string` | `'colors'` | Seção inicial |
| `exportFormat` | `ExportFormat` | `'json'` | Formato de exportação |
| `presets` | `object` | - | Presets de tema |
| `presetLabels` | `object` | - | Labels dos presets |
| `width` | `string \| number` | `'100%'` | Largura do editor |
| `height` | `string \| number` | `'600px'` | Altura do editor |
| `header` | `ReactNode` | - | Header customizado |
| `footer` | `ReactNode` | - | Footer customizado |
| `actions` | `ReactNode` | - | Actions customizadas |
| `title` | `string` | `'Theme Editor'` | Título do editor |
| `themeName` | `string` | `'my-theme'` | Nome do tema |
| `disabled` | `boolean` | `false` | Desabilitar editor |

### ExportFormat

```typescript
type ExportFormat = 'json' | 'typescript' | 'css' | 'mantine';
```

## Estrutura de Tema

O editor trabalha com a estrutura de tema do Mantine v7+:

```typescript
interface MantineThemeOverride {
  colors?: Record<string, string | Record<string, string>>;
  fontFamily?: string;
  fontSize?: string;
  lineHeight?: string;
  spacing?: Record<string, string>;
  radius?: Record<string, string>;
  // ... mais opções do Mantine
}
```
