# ArchbaseVisTimeline

Timeline interativa para visualização de eventos temporais, baseada em `vis-timeline` (MIT/Apache License).
Ideal para visualizar cronogramas, agendas históricas, gantt charts e séries temporais.

## Uso Básico

```tsx
import { ArchbaseVisTimeline } from '@archbase/components';

function App() {
  const items = [
    { id: 1, content: 'Projeto Alpha', start: '2024-01-01', end: '2024-01-15' },
    { id: 2, content: 'Projeto Beta', start: '2024-01-10', end: '2024-01-25' },
    { id: 3, content: 'Revisão', start: '2024-01-20', end: '2024-01-22' }
  ];

  return (
    <ArchbaseVisTimeline
      items={items}
      height={400}
    />
  );
}
```

### Opções de Orientação

- `'bottom'` - Timeline na parte inferior (padrão)
- `'top'` - Timeline na parte superior
- `'none'` - Sem timeline
- `'both'` - Timeline em ambas as partes

## Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `items` | `TimelineItem[]` | `[]` | Itens da timeline |
| `groups` | `TimelineGroup[]` | `[]` | Grupos dos itens |
| `start` | `Date \| string \| number` | - | Data inicial do range |
| `end` | `Date \| string \| number` | - | Data final do range |
| `range` | `[Date, Date]` | - | Range completo |
| `height` | `string \| number` | `'100%'` | Altura da timeline |
| `width` | `string \| number` | `'100%'` | Largura da timeline |
| `minHeight` | `string \| number` | `200` | Altura mínima |
| `orientation` | `TimelineOrientation` | `'bottom'` | Orientação da timeline |
| `editable` | `boolean` | `false` | Itens editáveis |
| `selectable` | `boolean` | `true` | Itens selecionáveis |
| `multiselect` | `boolean` | `true` | Seleção múltipla |
| `zoomable` | `boolean` | `true` | Habilitar zoom |
| `moveable` | `boolean` | `true` | Habilitar movimento |
| `showCurrentTime` | `boolean` | `true` | Mostrar tempo atual |
| `itemColor` | `string` | - | Cor dos itens |
| `groupColor` | `string` | - | Cor dos grupos |
| `backgroundColor` | `string` | - | Cor de fundo |
| `disabled` | `boolean` | `false` | Desabilitar interações |
| `noData` | `ReactNode` | - | Conteúdo quando não há itens |
| `onSelect` | `function` | - | Callback ao selecionar itens |
| `onRangeChange` | `function` | - | Callback ao mudar range |
| `onItemClick` | `function` | - | Callback ao clicar item |
| `onItemDoubleClick` | `function` | - | Callback ao duplo clique |
| `onItemUpdate` | `function` | - | Callback ao atualizar item |
| `timelineRef` | `function \| ref` | - | Ref da instância timeline |

### TimelineItem

```typescript
interface TimelineItem {
  id: string | number;
  content: string | ReactNode;
  start: Date | string | number;
  end?: Date | string | number;
  type?: 'box' | 'point' | 'range' | 'background';
  group?: string | number;
  className?: string;
  style?: CSSProperties;
  title?: string;
  editable?: boolean;
  data?: any;
}
```

### TimelineGroup

```typescript
interface TimelineGroup {
  id: string | number;
  content: string | ReactNode;
  order?: number;
  className?: string;
  style?: CSSProperties;
  title?: string;
}
```
