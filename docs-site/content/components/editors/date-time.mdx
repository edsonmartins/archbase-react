
# Data e Hora

Componentes para seleção de datas e horários.

## ArchbaseDatePickerEdit

Seleção de data com calendário.

### Exemplo

```tsx
import { ArchbaseDatePickerEdit } from '@archbase/components';

<ArchbaseDatePickerEdit
  dataSource={dataSource}
  dataField="dataNascimento"
  label="Data de Nascimento"
  placeholder="Selecione a data"
  disabled={isBrowsing}
/>
```

### Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `dataSource` | `ArchbaseDataSource<T>` | - | Fonte de dados |
| `dataField` | `string` | - | Campo do objeto |
| `label` | `string` | - | Rótulo do campo |
| `placeholder` | `string` | - | Texto placeholder |
| `minDate` | `Date` | - | Data mínima permitida |
| `maxDate` | `Date` | - | Data máxima permitida |
| `locale` | `string` | `'pt-BR'` | Localidade |
| `firstDayOfWeek` | `number` | `0` | Primeiro dia da semana (0=Dom) |
| `clearable` | `boolean` | `true` | Permite limpar data |
| `valueFormat` | `string` | `'DD/MM/YYYY'` | Formato de exibição |

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|

## ArchbaseMultiViewCalendar

Calendário multi-coluna para seleção de intervalos de datas com presets rápidos.

### Exemplo Básico

```tsx
import { ArchbaseMultiViewCalendar } from '@archbase/components';
import { useState } from 'react';

function Demo() {
  const [dateRange, setDateRange] = useState<[Date | null, Date | null]>([null, null]);

  return (
    <ArchbaseMultiViewCalendar
      label="Período do relatório"
      value={dateRange}
      onChange={setDateRange}
      showQuickPresets
      columns={2}
    />
  );
}
```

### Com Presets Personalizados

```tsx
const customPresets = [
  {
    label: 'Trimestre atual',
    getValue: () => {
      const now = new Date();
      const quarter = Math.floor(now.getMonth() / 3);
      const start = new Date(now.getFullYear(), quarter * 3, 1);
      const end = new Date(now.getFullYear(), (quarter + 1) * 3, 0);
      return [start, end];
    },
  },
  {
    label: 'Ano fiscal',
    getValue: () => {
      const year = new Date().getFullYear();
      return [new Date(year, 0, 1), new Date(year, 11, 31)];
    },
  },
];

<ArchbaseMultiViewCalendar
  label="Período"
  showQuickPresets
  quickPresets={customPresets}
/>
```

### Props

| Prop | Tipo | Padrão | Descrição |
|------|------|--------|-----------|
| `value` | `[Date \| null, Date \| null]` | - | Intervalo selecionado |
| `onChange` | `(value) => void` | - | Callback ao mudar |
| `columns` | `number` | `2` | Número de meses visíveis |
| `showQuickPresets` | `boolean` | `false` | Mostra presets rápidos |
| `quickPresets` | `ArchbaseQuickDatePreset[]` | - | Presets personalizados |
| `minDate` | `Date` | - | Data mínima selecionável |
| `maxDate` | `Date` | - | Data máxima selecionável |
| `clearable` | `boolean` | `true` | Permite limpar seleção |
| `locale` | `string` | `'pt-BR'` | Localidade do calendário |

### Presets Padrão

Quando `showQuickPresets` está ativo:

- Hoje
- Ontem
- Últimos 7 dias
- Últimos 30 dias
- Este mês
- Mês passado
- Este ano

## Formatos de Data

Os componentes de data suportam os seguintes formatos:

| Formato | Exemplo |
|---------|---------|
| `DD/MM/YYYY` | 25/12/2024 |
| `YYYY-MM-DD` | 2024-12-25 |
| `DD MMM YYYY` | 25 Dez 2024 |
| `MMMM YYYY` | Dezembro 2024 |
