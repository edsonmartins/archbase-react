
# Sistema de Layout

O pacote `@archbase/layout` fornece componentes para estruturação de interfaces.

## Componentes Disponíveis

| Componente | Descrição |
|------------|-----------|
| [ArchbaseCard](/layout/containers#archbasecard) | Card com header, body, footer |
| [ArchbaseContainer](/layout/containers#archbasecontainer) | Container responsivo |
| [ArchbaseSpace](/layout/space-system) | Sistema de posicionamento |
| [Dockable](/layout/dockable) | Janelas flutuantes |

## Instalação

```bash
pnpm add @archbase/layout
```
